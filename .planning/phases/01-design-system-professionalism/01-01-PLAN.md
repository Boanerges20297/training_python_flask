# Plan: 01-01 — Design Foundation

## Goal
Update the core design tokens and global styles to establish a professional, high-end baseline. This involves standardizing border radii, muting the color palette, and removing excessive visual noise (glows/heavy glassmorphism).

## Context
The user finds the current design "unprofessional" and "non-intuitive" due to extreme rounding (2.5rem in some places) and "scandalous" colors. We need to transition to a more sober, SaaS-like aesthetic (Zinc/Slate base) with a max radius of 0.75rem.

## Wave 1: Token Refactoring
- [ ] **Task 01-01-01**: Refactor `tokens.css` radii and shadows.
    - **Action**: 
        - Update `--radius-xs` to `0.25rem` (4px).
        - Update `--radius-sm` to `0.375rem` (6px).
        - Update `--radius-md` to `0.5rem` (8px).
        - Update `--radius-lg` to `0.625rem` (10px).
        - Update `--radius-xl` to `0.75rem` (12px).
        - Update `--radius-2xl` to `0.75rem` (12px) - capping the maximum.
        - Update shadows to be more subtle (less blur, lower opacity).
    - **Read First**: `frontend/src/assets/styles/tokens.css`
    - **Acceptance Criteria**: 
        - `tokens.css` contains `--radius-xl: 0.75rem;`.
        - `tokens.css` does not contain any radius value > `0.75rem`.

- [ ] **Task 01-01-02**: Mute Global Color Palette.
    - **Action**: 
        - Adjust `--color-primary` and variants to a more professional blue or slate-blue.
        - Update `--bg-primary` and `--bg-secondary` to use a consistent Zinc/Slate scale (e.g., #fafafa / #ffffff in light, #09090b / #18181b in dark).
        - Reduce the intensity of `--glass-blur` and `--glass-bg`.
    - **Read First**: `frontend/src/assets/styles/tokens.css`
    - **Acceptance Criteria**: 
        - Color tokens reflect a more muted, professional palette.

## Wave 2: Global CSS Cleanup
- [ ] **Task 01-01-03**: Clean up `index.css` and global resets.
    - **Action**: 
        - Remove any global "glow" effects or heavy gradients in `body` or main containers.
        - Ensure font rendering is optimized (`antialiased`).
    - **Read First**: `frontend/src/index.css`
    - **Acceptance Criteria**: 
        - No high-contrast radial gradients in `index.css`.
